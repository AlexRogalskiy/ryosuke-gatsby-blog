## Development

Running on GatsbyJS, an SSG that creates static React apps.

1. `npm install`
2. `npm run develop`


## Deployment

We locally build the files, then deploy using an NPM script that updates a specific Git repo branch called `gh-pages`.

`npm run deploy`
